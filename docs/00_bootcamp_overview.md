# 00_bootcamp_overview.md

## Goal

Learn how to use AI (Cursor) to build an end-to-end data project:

* EDA
* Feature engineering decisions integrated into EDA and modeling
* Modeling
* App (Streamlit)
* BI-ready outputs

The focus is not only the model, but **how to structure and validate AI-assisted work**.

---

## Workflow

For each task:

1. Read the task file in `docs/`
2. Ask the AI to propose a plan
3. Implement step by step
4. Validate against success criteria
5. Save outputs in the correct folders

---

## Key Principles

* Plan before coding
* Keep notebooks for exploration only
* Move reusable logic to `src/`
* Validate outputs explicitly
* Prefer simple solutions first

---

## Deliverables

By the end:

* EDA notebook with insights
* Trained churn model
* Streamlit app
* BI-ready tables and figures
* Executive summary
