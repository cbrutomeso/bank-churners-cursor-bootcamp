# Raw Data

Place the original Kaggle dataset in this folder.

Source:
[`https://www.kaggle.com/datasets/syviaw/bankchurners`](https://www.kaggle.com/datasets/syviaw/bankchurners)

Expected file:
* `data/raw/BankChurners.csv`

Notes:
* Treat this folder as read-only after downloading the source data.
* Do not commit the raw CSV to the repository.
* If the downloaded filename is different, rename it to `BankChurners.csv` or update the loading code explicitly.
